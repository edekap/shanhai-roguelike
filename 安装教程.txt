═══════════════════════════════════════
   【一】安卓安装教程
═══════════════════════════════════════

整个流程就 3 步，2 分钟搞定


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
第 1 步：找到下载好的 APK 文件
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

打开手机自带的「文件管理」app
（也叫「我的文件」「文件浏览器」等）

然后按你下载方式找文件：

▶ 微信收的：
  文件管理 → 内部存储 → Tencent → MicroMsg → Download
  找到「山海经肉鸽.apk」

▶ QQ收的：
  文件管理 → 内部存储 → Tencent → QQfile_recv
  找到「山海经肉鸽.apk」

▶ 浏览器下载的：
  文件管理 → 内部存储 → Download（下载）
  找到「山海经肉鸽.apk」

💡 找不到？在文件管理里搜索「山海经」或「.apk」


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
第 2 步：点击 APK 文件
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

找到「山海经肉鸽.apk」后，直接点它一下

如果是第一次安装 APK，系统会弹窗提示：
「为了您的安全，禁止安装未知来源应用」

这时点「设置」按钮 →
找到「允许从此来源安装应用」→ 打开开关 →
按返回键回到安装界面


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
第 3 步：点「安装」
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

系统会弹出安装界面，下方有「安装」按钮
点它 → 等几秒 → 显示「已安装」→ 点「打开」

游戏启动，可以玩了！


─────────────────────────────────────
  安卓常见问题
─────────────────────────────────────

❓ 点了 APK 没反应？
→ 先按第 2 步开「允许未知来源」权限，再点一次

❓ 提示「解析包失败」？
→ 下载过程文件可能损坏，重新下载一次

❓ 提示「应用未安装」？
→ 手机存储空间不够，清理一下内存

❓ 安装后找不到图标？
→ 桌面找「山海经肉鸽」图标，或在应用列表里找

❓ 之前开过权限还要再开吗？
→ 不用，第一次开过后永久生效

   【二】苹果用户方案
═══════════════════════════════════════

⚠️ 苹果手机（iPhone）不能安装 APK！

APK 是安卓系统的安装包格式，苹果 iOS 系统
完全不支持。如果你是苹果用户，请选以下方案：


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
方案 A：借一台安卓手机（最简单）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

找家人/朋友借一台安卓手机，
按上面的【安卓安装教程】安装即可。


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
方案 B：用电脑玩（推荐苹果用户）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

把「index.html」文件发到电脑上，
双击用浏览器（Chrome/Edge）打开即可玩。

操作方式：
  · W A S D 键 → 移动角色
  · 鼠标 → 瞄准方向
  · 左键 / 空格 → 射击
  · F 键 → 角色技能
  · R 键 → 死亡后重开


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
方案 C：网页版（需作者额外部署）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

如果作者把游戏部署到网上（比如 GitHub Pages），
苹果用户用 Safari 打开链接就能直接玩，
不用安装任何东西。

  · 用 Safari 打开游戏网址
  · 点击 Safari 底部的「分享」按钮
  · 选「添加到主屏幕」
  · 桌面会生成图标，像 App 一样打开

💡 这个方案需要作者额外操作，目前暂未部署。


═══════════════════════════════════════
   游戏玩法速览
═══════════════════════════════════════

📱 手机端操作（安卓）：
  · 左下虚拟摇杆 → 移动角色
  · 右下虚拟摇杆 → 瞄准 + 射击
  · 自动瞄准辅助，省心
  · 右下「技能」按钮 → 释放角色技能
  · 左上角「⏸」按钮 → 暂停/返回主菜单

🎮 建议横屏游玩，体验最佳

💻 电脑端操作：
  · W A S D → 移动
  · 鼠标 → 瞄准
  · 左键/空格 → 射击
  · F → 技能
  · R → 重开
